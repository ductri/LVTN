Instructions:
1. Open metamap server and cmd at this folder
2. Run in cmd: java -jar metamapNgram.jar
3. Run in cmd: javac CallKit.java
4. Run in cmd: java CallKit negex_triggers.txt ngram.txt


===============================================

Folder includes some basic parts:
1. input.xls: ID & Sentence
2. metamapNgram.jar: get ngram from sentences of input.xls, output will be ngram.txt
3. CallKit.java: get phrases and sentence from ngram.txt, negation output will be stored in result.txt
